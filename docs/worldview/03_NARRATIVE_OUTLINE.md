# 因果修真世界 - 核心叙事大纲

## 一、叙事核心主题

### 1.1 核心主题
「因果修真世界」的核心叙事主题是：**命运无常，因果循环；逆天改命，终成大道。**

### 1.2 叙事哲学
- **无常**：命运不可预测，即使功德圆满也可能遭遇不幸
- **因果**：行为决定未来，跨越轮回的因果链条
- **抗争**：即使命运无常，也要努力抗争
- **超脱**：最终超脱轮回，达到逍遥境界

### 2.2 第一幕：凡尘入世

#### 场景1：出生
- **地点**：人界某个小村庄
- **事件**：玩家出生，开始人生
- **选择**：家族背景（随机/选择）

#### 场景2：觉醒
- **地点**：家族/宗门
- **事件**：展现修炼天赋，踏上修真之路
- **选择**：修炼方向（剑修/法修/体修等）

#### 场景3：历练
- **地点**：新手村周边
- **事件**：首次战斗，初步体验因果系统
- **选择**：功业与路径选择初体验（无任务引导）

#### 场景4：入门
- **地点**：修真宗门
- **事件**：加入宗门，正式开始修炼
- **选择**：选择宗门/师父

### 2.3 第二幕：纵横六界

#### 场景5：成长
- **地点**：宗门/秘境
- **事件**：修为提升，获得机缘
- **选择**：资源分配、修炼路线

#### 场景6：结交
- **地点**：各大城市
- **事件**：结识NPC，建立羁绊
- **选择**：结交朋友/树立仇敌

#### 场景7：纷争
- **地点**：六界各地
- **事件**：卷入仙魔大战/宗门纷争
- **选择**：立场选择

#### 场景8：探索
- **地点**：秘境/禁地
- **事件**：探索未知，获得奇遇
- **选择**：冒险决策

### 2.4 第三幕：天道审判

#### 场景9：巅峰
- **地点**：仙界/神界
- **事件**：达到修为巅峰，面临天劫
- **选择**：渡劫/归隐

#### 场景10：审判
- **地点**：冥界/天道
- **事件**：死亡后接受天道审判
- **机制**：功德结算 + 随机判定

#### 场景11：轮回
- **地点**：轮回池
- **事件**：在功德允许的投胎池内 **自行选择** 下一世身份
- **机制**：功德结算 → 生成可选池 → 玩家选择 → 六道轮回落地（单命制：仅一条活跃生命）

## 五、叙事原则

### 5.1 核心原则
1. **无常性**：命运不可预测，即使准备充分也可能失败
2. **因果性**：所有行为都有后果，跨越轮回
3. **多样性**：多种结局，没有唯一正确路线
4. **情感性**：重视与NPC的情感连接

### 5.2 禁忌原则
1. 禁止「注定成功」的叙事
2. 禁止「善恶单轴简单奖惩」的叙事
3. 禁止「任务清单驱动」的主线叙事
3. 禁止「线性单一」的叙事
4. 禁止「完美主角」的设定

## 六、结局设定

### 6.1 结局类型

| 结局类型 | 条件 | 描述 |
|----------|------|------|
| **成仙结局** | 功德极高+机缘 | 飞升仙界，跳出轮回 |
| **逍遥结局** | 功德中等+选择归隐 | 逍遥人间，长生不老 |
| **轮回结局** | 功德普通 | 继续轮回，积累功德 |
| **堕落结局** | 功德极低 | 堕入魔道/地狱 |
| **传奇结局** | 特殊条件 | 成为传说，影响后世 |

### 6.2 结局影响
- 结局影响下一世的初始条件
- 结局影响世界演化
- 结局影响NPC命运
