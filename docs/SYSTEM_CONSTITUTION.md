# 因果修真世界 - 开发宪法

> **效力**：本文档约束一切设计、开发与 AI 辅助编码。冲突时以本文档为准。  
> **版本**：v1.0 | 2026-06-12

---

## 一、产品定义

**因果修真世界** 是 **人生模拟** 游戏，不是营利型修真手游。

### 1.1 必须做到

1. 体验生命：生老病死、生存压力、选择有重量
2. 命运与因果：随机中有逻辑，后果可学习
3. 单命制：任意时刻仅一条活跃生命
4. 玩家自由：从商/从政/修真/为寇等路径本就在，弱引导
5. MVP 可玩闭环：`活一世 → 死 → 选胎 → 再活一世`

### 1.2 禁止做到

1. 每日任务、主线任务清单、任务 API
2. 固定技能树作为玩家成长主轴
3. 玩家可见的善恶值/功德数值面板
4. 多命并行、区服选出生
5. 玩家言出法随、封神、至高传承
6. 被恶意杀害的投胎加权补偿
7. 以营利和留存率为首要设计目标

---

## 二、游戏主轴

```
经历 → 感悟 → 神通 → 影响世界 → 轮回
```

核心循环：

```
生存 → 事件 → 选择 → 因果反馈 → 死亡 → 选胎 → 新一世
```

修真是 **分支路径**，不是默认玩法。

---

## 三、硬性规则

| 规则 | 说明 |
|------|------|
| **单命制** | `active_life_id` 唯一；轮回结算前不得创建新生命 |
| **投胎池** | 功德 + 判定值定池边界；玩家在池内自选 |
| **判定值** | 只扩缩池，不替玩家抽终点 |
| **自杀** | 先 -500 功德；清空记忆；**保留印记**；池内自选 |
| **恶意杀害** | 不加权投胎；仍按功德定池 |
| **鬼修** | 仅死后进入，不作初始种族 |
| **时间** | 现实 1 年 ≈ 游戏 100 年（1:10） |
| **世界** | 单宇宙、无区服；出生随机不可选 |
| **初始世界** | 无王朝；约 500 NPC 散落各行业 |

---

## 四、因果与 UI

- **功德**：跨世账本，决定投胎池
- **业力**：当世文明路径权重（杀/霸/武/王/魔等）
- **UI**：模糊风险提示（「可能记恨」「后果难料」）；数值仅后台

---

## 五、天道权限

| 能力 | 归属 |
|------|------|
| 言出法随（调权重、投事件） | **仅开发者** |
| 封神 | **仅开发者** |
| 至高心法传承 | **仅开发者** |
| 受限子世界 | 玩家 Phase 2；不能再诞子世界 |

开发者可调权重与投事件，**不直接篡改**单个玩家的结算结果。

---

## 六、MVP 范围

### 纳入 MVP

- 单命制 + 选胎 UI
- 事件驱动人生（无任务）
- 五感系统
- 完整距离战斗
- Tick + 状态机世界模拟

### 不纳入 MVP

- `operation/*` 商业化运营
- 玩家创世 / 改主世界规则
- 百万并发 / GPT 全 NPC
