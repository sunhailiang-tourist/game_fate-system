# 因果修真世界 - 游戏开发文档目录

## 项目概述
「因果修真世界」是一款具备颠覆性创新的修真游戏，核心特色包括：
- 命运无常的天道轮回系统
- 真实的生命体验（生老病死）
- 五感共鸣的沉浸体验
- 情感羁绊的深度社交
- 玩家共创的活世界
