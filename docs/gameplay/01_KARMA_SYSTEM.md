# 因果修真世界 - 因果系统设计

## 一、系统概述

### 1.1 设计目标
因果系统是「因果修真世界」的核心玩法系统，旨在实现：
- **行为→结果**的延迟反馈机制
- **跨越轮回**的因果链条
- **真实感**的命运无常体验

### 1.2 核心机制

```
玩家行为 → 因果节点生成 → 延迟反馈 → 世界变化
```

### 1.3 功德与业力双轴（取代善恶值面板）

| 轴 | 作用域 | 玩家可见 | 说明 |
|----|--------|----------|------|
| **功德** | 跨世 | 模糊档位（善人/义士等） | 决定投胎池边界；继承 30% |
| **业力** | 当世 | **不显示数值** | 文明路径权重（杀/霸/武/王/魔等）；影响事件概率与世界反应 |

> **废除**：玩家 HUD 上的「善恶值」「因果条 ± 数值」。UI 仅显示模糊风险提示，如「可能记恨」「后果难料」「天道或将记一笔」。

## 二、因果链机制

### 2.1 因果链结构

```
行为事件 ──→ 因果节点 ──→ 延迟效应 ──→ 结果反馈
    │              │              │              │
    ↓              ↓              ↓              ↓
  触发时间      生成时间      触发时间      反馈时间
```

### 2.2 因果节点类型

| 节点类型 | 触发条件 | 延迟时间 | 影响范围 |
|----------|----------|----------|----------|
| **即时因果** | 直接行为 | 0-1分钟 | 个人 |
| **短期因果** | 重要行为 | 10-30分钟 | 个人/小范围 |
| **中期因果** | 重大行为 | 1-2小时 | 区域 |
| **长期因果** | 决定性行为 | 24小时以上 | 世界 |
| **跨轮回因果** | 累积行为 | 下一世 | 轮回 |

### 2.3 因果值计算

**因果值 = 行为价值 × 影响系数 × 时间衰减**

| 行为类型 | 行为价值 | 影响系数 | 时间衰减 |
|----------|----------|----------|----------|
| 轻微善举 | +1-5 | 1.0 | 快 |
| 中等善举 | +10-20 | 1.5 | 中 |
| 重大善举 | +50-100 | 2.0 | 慢 |
| 轻微恶行 | -1-5 | 1.0 | 快 |
| 中等恶行 | -10-20 | 1.5 | 中 |
| 重大恶行 | -50-100 | 2.0 | 慢 |

| 事件类型 | 行为描述 | 功德值 | 难度 |

| **守护一方** | 建立保护区域，保护平民 | +200 | 极高 |
| **拯救苍生** | 阻止重大灾难（如瘟疫、洪水） | +500 | 极高 |
| **普渡众生** | 大规模救助生灵 | +1000 | 神话 |

#### 2.4.2 功德消耗事件

## 三、因果事件系统

### 3.1 事件触发机制

```
玩家行为 → 事件检测 → 因果判定 → 节点生成 → 延迟执行
```

### 3.2 事件类型

| 事件类型 | 描述 | 例子 |
|----------|------|------|
| **功业事件** | 影响功德或业力 | 救人/屠戮证道 |
| **机缘事件** | 获得稀有资源 | 发现秘境 |
| **灾难事件** | 遭受损失 | 遭遇伏击 |
| **关系事件** | 改变NPC关系 | 结交/背叛 |
| **世界事件** | 影响世界状态 | 改变区域属性 |

### 3.3 事件概率

**事件概率 = 基础概率 × 因果值修正 × 随机因子**

| 事件类型 | 基础概率 | 因果值修正 |
|----------|----------|----------|
| 善缘事件 | 30% | 因果值>0时+10%/100点 |
| 恶缘事件 | 30% | 因果值<0时+10%/-100点 |
| 随机事件 | 40% | 不受因果值影响 |

## 四、因果可视化（模糊 UI）

### 4.1 因果图谱

```
玩家
  │
  ├──→ 救死扶伤 ──→ 获得感激 ──→ 功德↑
  │
  ├──→ 屠戮证道 ──→ 魔道标记 ──→ 业力↑，事件链变化
  │
  └──→ 探索秘境 ──→ 获得机缘 ──→ 随机奖励
```

### 4.2 玩家可见信息

| 显示方式 | 内容 | 位置 |
|----------|------|------|
| **天命状态** | 默默无闻 → 小有名气 → 天命加身（非数值） | 角色概览 |
| **风险提示** | 「可能记恨」「后果难料」等模糊文案 | 抉择弹窗 |
| **事件日志** | 叙事化因果反馈 | 日志面板 |
| **后台隐藏** | 功德精确值、业力权重、判定值 | 仅服务端 |

## 五、跨轮回因果

### 5.1 因果传承机制

```
当前轮回行为 → 因果积累 → 轮回结算 → 下一世继承
```

### 5.2 继承规则

| 继承类型 | 继承比例 | 说明 |
|----------|----------|------|
| **功德继承** | 30% | 直接影响下一世初始功德 |
| **因果印记** | 100% | 特殊因果事件永久记录 |
| **命运倾向** | 50% | 影响随机事件概率 |

### 5.3 因果印记

| 印记类型 | 获取条件 | 效果 | 持续时间 |
|----------|----------|------|----------|
| **善人之印** | 累积1000功德 | 善缘事件+20% | 永久 |
| **恶人之印** | 累积-1000功德 | 恶缘事件+20% | 永久 |
| **天命之人** | 特殊机缘 | 奇遇概率+30% | 永久 |
| **灾祸之身** | 连续不幸 | 灾难概率+20% | 3轮回 |

## 六、系统约束

### 6.1 因果值范围
- **最小值**：-10000
- **最大值**：10000
- **初始值**：0

### 6.2 因果衰减规则
- **每日衰减**：当前因果值 × 1%
- **衰减下限**：-1000 / +1000
- **特殊保护**：因果印记不受衰减影响

### 6.3 系统边界
- 因果值不能直接决定轮回结果（仅作为加权）
- 单次行为因果值变化有上限（±100）
- 跨轮回继承有上限（±3000）

## 七、与其他系统交互

### 7.1 与轮回系统
- **判定值**仅扩缩投胎池边界，不替代玩家在池内的选择
- 因果印记 100% 跨世继承；功德 30% 继承
- **功德值决定投胎可选池**：玩家只能在功德解锁的轮回道/物种/身份档位内自选（见轮回系统 4.2.1）
- **自杀轮回**：先 -500 功德，清空记忆、**保留印记**，再按扣减后功德生成可选池
- **恶意杀害**：不加权投胎补偿，仍按功德定池
- **单命制**：任意时刻仅一条活跃生命，轮回结算完成后方可开启下一世

### 7.2 与生命体验系统
- 因果值影响年龄变化
- 因果值影响悟性提升

### 7.3 与五感共鸣系统
- 高因果值增强感知能力
- 低因果值增加感知失误概率

### 7.4 与情感羁绊系统
- 因果值影响NPC初始好感度
- 特殊因果事件改变羁绊关系

## 八、开发注意事项

### 8.1 数据存储
- 因果节点需要持久化存储
- 跨轮回数据需要独立保存

### 8.2 性能优化
- 因果计算需要异步执行
- 大量因果节点需要清理机制

### 8.3 测试要点
- 测试因果链的延迟反馈
- 测试跨轮回继承机制
- 测试概率系统的公平性
