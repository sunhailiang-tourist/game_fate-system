# 因果修真世界 - 战斗系统设计

---

## 一、系统概述

### 1.0 MVP 范围

**MVP 包含完整距离战斗体系**（近战/中距/远程/超远），不作为 Phase 2 删减项。战斗服务人生模拟中的生存、冲突与文明路径（武/杀/魔等），而非独立刷本玩法。

### 1.1 设计目标
战斗系统旨在实现：
- **距离策略**：不同修炼方向有不同的有效战斗距离
- **职业特色**：每个修炼方向有独特的战斗机制
- **安全意识**：玩家需要时刻保持安全距离
- **战术深度**：距离控制成为核心战斗策略
- **万灵通用**：所有生灵适用统一的战斗距离体系

### 1.2 核心机制

```
战斗距离 → 修炼方向加成 → 技能效果 → 战斗结果
    ↓           ↓            ↓           ↓
  距离计算   方向修正     技能判定    伤害计算
```

### 1.3 适用范围

| 生灵类型 | 适用范围 | 说明 |
|----------|----------|------|
| **人族** | 修士/凡人 | 修仙者适用修士战斗规则，凡人适用野兽规则 |
| **妖族** | 兽类/精怪 | 继承野兽本能的战斗方式 |
| **魔族** | 魔族/魔兽 | 魔族特有战斗特性 |
| **鬼族** | 鬼魂/厉鬼 | 灵体战斗特性 |
| **神兽** | 先天神兽 | 超越普通野兽的战斗能力 |
| **灵兽** | 开启灵智的兽类 | 具备修炼潜力 |

## 四、距离战斗修正系统

### 4.1 距离衰减公式

```
最终伤害 = 基础伤害 × 距离修正 × 方向加成 × 其他修正

距离修正 = 最佳距离伤害 / 实际距离
```

### 4.2 距离修正表

|-------------------|----------|
| 100% | 1.0 |
| 150% | 0.8 |
| 200% | 0.6 |
| 300% | 0.4 |
| 500% | 0.2 |
| 1000% | 0.1 |

| 危险信号 | 说明 | 应对措施 |

## 六、战斗属性计算

### 6.1 基础属性

| 属性 | 说明 | 影响 |
|------|------|------|
| **攻击力** | 造成伤害的基础值 | 伤害计算 |
| **防御力** | 减免伤害的基础值 | 减伤计算 |
| **速度** | 行动先后顺序 | 先手优势 |
| **生命值** | 存活能力 | 生存能力 |
| **灵力值** | 技能消耗 | 续航能力 |

### 6.2 战斗伤害公式

```
最终伤害 = (攻击方攻击力 - 防御方防御力 × 防御系数) × 距离修正 × 方向加成 × 技能加成 × 暴击修正

暴击伤害 = 最终伤害 × 2.0
```

### 6.3 防御系数

| 防御类型 | 防御系数 |
|----------|----------|
| 物理防御 | 1.0 |
| 法术防御 | 0.8 |
| 毒抗 | 0.6 |
| 元素抗性 | 0.7 |

## 七、战斗流程

### 7.1 战斗阶段

```
遭遇 → 对峙 → 试探 → 交锋 → 决胜 → 结束
  ↓      ↓       ↓      ↓       ↓       ↓
 发现    拉开距离  技能   拉扯   爆发   胜败
```

### 7.2 行动优先级

| 优先级 | 行动类型 | 说明 |
|--------|----------|------|
| 1 | 位移技能 | 调整距离 |
| 2 | 控制技能 | 先手控制 |
| 3 | 核心输出 | 主要伤害 |
| 4 | 辅助技能 | 增益/治疗 |
| 5 | 普通攻击 | 补充伤害 |

### 7.3 战斗结束

| 结果 | 条件 | 惩罚 |
|------|------|------|
| **胜利** | 对方生命值归零 | 获得奖励 |
| **失败** | 己方生命值归零 | 修为损失 |
| **逃跑** | 成功脱离战斗 | 无惩罚 |
| **平局** | 双方都存活 | 无奖励 |

| **神兽** | 0-100年 | 101-500年 | 501-2000年 | 2000年+ |

### 9.3 发育对战斗的影响

| 法修过于强大 | 增加突进职业，提高吟唱时间 |
