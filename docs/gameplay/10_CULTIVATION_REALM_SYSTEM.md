# 因果修真世界 - 境界体系系统

## 一、系统概述

### 1.1 设计目标
境界体系系统旨在实现：
- **完整的修炼境界划分**：从凡人到成仙的完整路径
- **天道规则约束**：修行者不得参与人间争斗、不得伤害平民
- **天道标记机制**：伤害平民会被天道标记，增加渡劫失败概率
- **渡劫系统**：每个大境界突破需要渡劫

### 1.2 核心机制

```
凡人 → 炼气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 成仙
    ↓      ↓      ↓      ↓      ↓      ↓      ↓      ↓
 凡俗   入门    打基础  结金丹  元婴出  元神化  渡天劫  登仙境
```

#### 2.8.2 成神共同点
- **法则领悟**：必须领悟本道本源法则达到100%
- **悟性圆满**：悟性属性达到上限
- **特殊条件**：各道有独特的成神要求

#### 2.8.3 成神后能力
- **虚空造物**：所有成神者均可凭空创造物质
- **法则化身**：化身为法则本身
- **世界投影**：在其他世界显现化身
- **创造世界**：道祖级别可创造独立小世界

#### 2.8.4 神位唯一性法则

| 规则 | 说明 |
|------|------|
| **唯一性** | 每个元素/修行方向**只有一个至高神位** |
| **神位限制** | 同一元素道祖只能有一位（如庚金道祖只能有一个） |
| **神位争夺** | 挑战现任道祖成功可继承神位 |
| **死亡回归** | 神位持有者死亡后，**元气消散** |
| **道果回归** | 如果道果未被夺走吸收，**回归世界本源** |

```
道果掉落 → 未被拾取 → 开始消散倒计时 → 完全消散 → 回归世界本源
    ↓                ↓                  ↓            ↓
 击杀目标        1小时内             24小时后     资源恢复
```

| 道果类型 | 消散时间 | 回归本源量 |
|----------|----------|------------|
| **金丹** | 24小时 | 1000灵力 |
| **元婴** | 12小时 | 5000灵力 |
| **元神** | 6小时 | 10000灵力 |
| **道果** | 1小时 | 50000灵力 |

### 2.9 资源恒定法则

#### 2.9.1 核心公式

```
世界总资源 = 初始资源 + 道果回归量 - 资源消耗量

当神位持有者死亡且道果未被吸收时：
- 道果能量回归世界本源
- 本源能量可重新生成资源
- 维持世界资源总量恒定
```

#### 2.9.2 资源循环流程

```
修炼者产生道果 → 道果被吸收/消散 → 回归世界本源 → 生成新资源 → 新修炼者获取资源
    ↓                     ↓                   ↓                  ↓
  金丹/元婴         能量转移           能量回归            资源再生
```

#### 2.9.3 天道平衡机制

| 行为 | 天道反应 | 平衡措施 |
|------|----------|----------|
| 过度吸收 | 天道标记增加 | 渡劫难度提升 |
| 道果浪费 | 资源回归 | 世界资源补充 |
| 神位空缺 | 法则弱化 | 新神位候选人出现 |
| 资源枯竭 | 再生加速 | 新资源点生成 |

### 2.2 各境界详细说明

#### 2.2.1 凡人境（0级）
- **状态**：普通人类，无法感应灵气
- **特点**：受生老病死束缚
- **突破方式**：获得修炼功法，开启灵根

#### 2.2.2 炼气期（1级）
- **阶段**：炼气一层至十层
- **能力**：引气入体，初步修炼
- **标志**：体内产生微弱灵力
- **突破需求**：灵力积累到临界点

#### 2.2.3 筑基期（2级）
- **阶段**：筑基初期、中期、后期
- **能力**：打下修炼基础，灵力稳固
- **标志**：筑基成功，寿元翻倍
- **突破需求**：感悟天地法则

#### 2.2.4 金丹期（3级）
- **阶段**：金丹初期、中期、后期、圆满
- **能力**：凝结金丹，灵力质变
- **标志**：丹田内形成金丹
- **突破需求**：金丹圆满，感悟生死

#### 2.2.5 元婴期（4级）
- **阶段**：元婴初期、中期、后期、圆满
- **能力**：元婴出窍，长生不老
- **标志**：元婴成形，可离体活动
- **突破需求**：元婴化神，领悟空间法则

#### 2.2.6 化神期（5级）
- **阶段**：化神初期、中期、后期、圆满
- **能力**：元神化形，神游太虚
- **标志**：元神脱离肉身独立存在
- **突破需求**：感悟天道，积累功德

#### 2.2.7 渡劫期（6级）
- **阶段**：渡劫初期、中期、后期
- **能力**：引动天劫，准备飞升
- **标志**：天地感应，劫云汇聚
- **突破需求**：渡过九重天劫

#### 2.2.8 成仙（7级）
- **状态**：飞升仙界
- **能力**：超越轮回，永恒不灭
- **标志**：脱离凡界，进入仙界

#### 4.1.2 渡劫成功率计算公式

```
渡劫成功率 = 基础成功率 + 功德修正 - 天道标记惩罚 - 随机因子

基础成功率: 50%
功德修正: 每1000功德 +5%（最高+30%）
天道标记惩罚: 每标记 -10%
随机因子: ±10%
```

#### 4.1.3 渡劫失败后果

| 失败程度 | 后果 |
|----------|------|
| **轻伤** | 修为下降一级 |
| **重伤** | 修为下降两级，寿命-100年 |
| **陨落** | 直接死亡，进入轮回 |

## 八、系统约束

### 8.1 修行者行为约束
- **禁止伤害平民**：攻击未修炼者会触发天道标记
- **禁止大规模破坏**：破坏城市、村庄会触发天道标记
- **禁止滥用力量**：对凡人使用超出其承受范围的力量会触发标记

### 8.2 突破约束
- **功德要求**：化神期突破需要至少1000功德
- **标记限制**：成仙突破需要天道标记<3
- **资源要求**：每个突破阶段需要相应丹药
