# 因果修真世界 - 轮回系统设计

## 一、系统概述

### 1.1 设计目标
轮回系统是「因果修真世界」的核心玩法系统，旨在实现：
- **命运无常**的真实体验
- **六道轮回**的完整机制
- **功德影响**但不决定的设计

### 1.2 核心机制

```
死亡 → 灵魂审判 → 功德结算 → 随机判定 → 轮回分配 → 重生
```

---

## 二、死亡机制

### 2.1 死亡触发条件

| 死亡类型 | 触发条件 | 例子 |
|----------|----------|------|
| **生命值归零** | 战斗失败 | 被怪物击杀 |
| **自然死亡** | 年龄达到上限 | 寿终正寝 |
| **意外死亡** | 特殊事件 | 跌落悬崖 |
| **渡劫失败** | 渡劫失败 | 天劫击杀 |

### 2.2 死亡惩罚
- **损失当前轮回的所有修为**（保留因果印记）
- **损失部分财产**（30%随机掉落）
- **进入冥界等待审判**

---

## 三、天道审判

### 3.1 审判流程

```
进入冥界 → 功德结算 → 因果印记检测 → 随机判定 → 轮回分配
```

### 3.2 功德结算

| 功德等级 | 范围 | 描述 | 加权系数 |
|----------|------|------|----------|
| **大善** | 5000+ | 积德行善，造福苍生 | +30% |
| **善** | 1000-4999 | 多行不义必自毙 | +15% |
| **中立** | -999-999 | 无功无过 | 0% |
| **恶** | -4999--1000 | 作恶多端 | -15% |
| **大恶** | -5000以下 | 罪大恶极 | -30% |

### 3.3 随机判定机制

**最终判定值 = 基础随机数(0-100) + 功德加权 + 因果印记修正**

| 随机数范围 | 结果等级 | 说明 |
|------------|----------|------|
| 0-19 | 极差 | 即使功德高也可能出现 |
| 20-49 | 较差 | 中等功德常见结果 |
| 50-79 | 普通 | 标准结果 |
| 80-99 | 极佳 | 即使功德低也可能出现 |

## 四、六道轮回

### 4.1 六道类型

| 轮回道 | 类型 | 功德要求 | 特点 |
|--------|------|----------|------|
| **天道** | 出生 | 大善+特殊机缘 | 直接成仙，超越轮回 |
| **人道** | 出生 | 善/中立 | 正常人生，可修炼 |
| **畜生道** | 出生 | 恶/中立 | 野兽形态，记忆保留片段 |
| **植物道** | 出生 | 中立+随机 | 灵植形态，千年化形 |
| **饿鬼道** | 死亡后 | 大恶 | 永久饥饿，痛苦轮回 |
| **地狱道** | 死亡后 | 大恶+随机 | 受刑N年，痛苦至极 |

### 4.2 轮回分配规则

```
最终判定值 → 轮回道选择 → 物种随机 → 身份生成 → 属性继承
```

| 判定值范围 | 可能轮回道 | 概率分布 |
|------------|----------|----------|
| 0-19 | 畜生道/饿鬼道/地狱道 | 畜生道40% / 饿鬼道30% / 地狱道30% |
| 20-49 | 畜生道/人道/植物道 | 畜生道40% / 人道40% / 植物道20% |
| 50-79 | 人道/植物道/仙界候补 | 人道85% / 植物道10% / 仙界候补5% |
| 80-99 | 人道/仙界/天道 | 人道60% / 仙界30% / 天道10% |

### 4.3 特殊轮回

| 特殊轮回 | 触发条件 | 效果 |
|----------|----------|------|
| **天道飞升** | 判定值95+且功德≥5000 | 直接成仙 |
| **地狱受刑** | 判定值5-且功德≤-5000 | 受刑10年游戏时间 |
| **植物化形** | 植物道轮回满千年 | 可化为人形 |
| **饿鬼解脱** | 饿鬼道中积满善功 | 可转世为人 |

### 4.6 初始属性生成公式

```
最终属性 = 基础属性 + 物种加成 + 功德继承 + 随机波动

基础属性: 根据物种设定的基准值
物种加成: 物种特有的属性加成（如灵兽灵力+15%）
功德继承: 上一世功德的30%转化为悟性
随机波动: ±10%的随机值
```

### 4.7 初始年龄与寿元随机

#### 4.7.1 初始年龄随机

| 轮回道 | 初始年龄范围 | 概率分布 |
|--------|--------------|----------|
| **人道** | 0-10岁 | 均匀分布 |
| **畜生道** | 0-5岁（幼年期） | 均匀分布 |
| **植物道** | 0-100年 | 均匀分布 |

#### 4.7.2 寿元随机机制

```
初始寿元 = 物种基础寿元 × (1 + 功德修正 + 随机波动)

物种基础寿元: 凡人100岁, 灵兽500岁, 千年古树1000年
功德修正: 每1000功德 +5%寿元（最高+20%）
随机波动: ±20%
```

| 物种 | 基础寿元 | 寿元范围 |
|------|----------|----------|
| **普通人类** | 100岁 | 80-140岁 |
| **妖族混血** | 150岁 | 120-210岁 |
| **灵体转世** | 200岁 | 160-280岁 |
| **谪仙人** | 500岁 | 400-700岁 |
| **普通野兽** | 50岁 | 40-70岁 |
| **灵兽** | 500岁 | 400-700岁 |
| **神兽后裔** | 1000岁 | 800-1400岁 |
| **普通灵草** | 500年 | 400-700年 |
| **千年古树** | 1000年 | 800-1400年 |
| **先天仙药** | 3000年 | 2400-4200年 |

## 五、重生机制

### 5.1 身份生成

```
轮回道确定 → 身份随机生成 → 初始属性分配
```

### 5.2 属性继承

| 继承类型 | 继承内容 | 继承比例 |
|----------|----------|----------|
| **功德继承** | 上一世功德 | 30% |
| **因果印记** | 特殊印记 | 100% |
| **天赋继承** | 特殊天赋 | 随机50% |
| **记忆碎片** | 前世记忆 | 随机触发 |

### 5.3 重生保护期
- **新手保护**：前30分钟不会被攻击
- **属性保护**：初始属性有保底

## 七、系统约束

### 7.1 轮回限制
- **最小轮回间隔**：1分钟
- **最大轮回次数**：无限制
- **特殊轮回冷却**：天道飞升后10轮回冷却

### 7.2 功德范围
- **最小值**：-10000
- **最大值**：10000
- **衰减规则**：每日衰减1%

### 7.3 随机公平性
- 使用加密随机数生成器
- 确保概率分布均匀
- 防止作弊

---

## 八、与其他系统交互

### 8.1 与因果系统
- 因果值影响功德结算
- 因果印记继承

### 8.2 与生命体验系统
- 轮回道影响初始年龄
- 轮回道影响属性成长

### 8.3 与五感共鸣系统
- 不同轮回道感知能力不同
- 植物道有特殊感知

### 8.4 与情感羁绊系统
- 重要NPC关系跨轮回记忆
- 羁绊值影响轮回加权

## 九、开发注意事项

### 9.1 数据存储
- 轮回记录需要持久化
- 因果印记需要独立保存

- 审判流程异步执行
- 大量轮回数据需要清理

### 9.3 测试要点
- 测试随机判定的公平性
- 测试功德加权机制
- 测试跨轮回继承
