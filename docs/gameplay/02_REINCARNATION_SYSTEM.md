# 因果修真世界 - 轮回系统设计

## 一、系统概述

### 1.1 设计目标
轮回系统是「因果修真世界」的核心玩法系统，旨在实现：
- **命运无常**的真实体验
- **六道轮回**的完整机制
- **功德影响**但不决定的设计
- **单命制**：任意时刻仅存在一条活跃生命，禁止任何形式的并行人生
- **功德门槛投胎**：下一世进入时，可选投胎范围由功德决定，玩家在池内自选

### 1.2 核心机制

```
死亡 → （可选魂修短阶段）→ 灵魂审判 → 功德结算 → 判定值扩缩池 → 生成可选投胎池 → 玩家选择 → 重生
```

### 1.3 单命制（硬性规则）

| 规则 | 说明 |
|------|------|
| **一命当前** | 账号在任意时刻只能有 **1 条** 活跃生命 |
| **禁止并行** | 不得同时操控多个角色、多条人生线、多个并行存档 |
| **轮回门禁** | 仅当当前生命 **彻底终结**（死亡并完成轮回结算）后，方可开启下一世 |
| **世界唯一性** | 同一账号不会在世界中同时存在多个实体 |

> 设计意图：强化「这一条命」的重量；玩家体验的是连续的灵魂历程，而非多开刷开局。

## 二、死亡机制

### 2.1 死亡触发条件

| 死亡类型 | 触发条件 | 例子 |
|----------|----------|------|
| **生命值归零** | 战斗失败 | 被怪物击杀 |
| **自然死亡** | 年龄达到上限 | 寿终正寝 |
| **意外死亡** | 特殊事件 | 跌落悬崖 |
| **渡劫失败** | 渡劫失败 | 天劫击杀 |

### 2.2 死亡惩罚
- **损失当前轮回的所有修为**（保留因果印记）
- **损失部分财产**（30%随机掉落）
- **进入冥界等待审判**

## 三、天道审判

### 3.1 审判流程

```
进入冥界 → 功德结算 → 因果印记检测 → 随机判定（加权）→ 生成可选投胎池 → 玩家选择 → 轮回分配
```

> **非自杀死亡**：玩家在功德允许的投胎池内 **自行选择** 下一世（轮回道 / 物种 / 身份等可选项受功德限制）。  
> **自杀死亡**：先执行功德削减，再在削减后的可选池内选择投胎（见生命体验系统「自杀机制」）。

### 3.2 功德结算

| 功德等级 | 范围 | 描述 | 加权系数 |
|----------|------|------|----------|
| **大善** | 5000+ | 积德行善，造福苍生 | +30% |
| **善** | 1000-4999 | 多行不义必自毙 | +15% |
| **中立** | -999-999 | 无功无过 | 0% |
| **恶** | -4999--1000 | 作恶多端 | -15% |
| **大恶** | -5000以下 | 罪大恶极 | -30% |

### 3.3 判定值机制（仅扩缩投胎池）

**判定值 = 基础随机数(0-100) + 功德加权 + 因果印记修正**

> **优先级**：判定值 **只决定投胎池的上下边界**，**不替代**玩家在池内的最终选择。系统不为玩家抽取「终点投胎结果」。

| 判定值范围 | 池边界效果 | 说明 |
|------------|------------|------|
| 0-19 | 池子极窄 | 即使功德高，边界也可能收紧 |
| 20-49 | 池子偏窄 | 中等功德常见 |
| 50-79 | 标准池 | 按功德表过滤 |
| 80-99 | 池子放宽 | 可触及更高档位选项 |

### 3.4 魂修阶段（可选）

死亡后、投胎 UI 之前，玩家可进入 **短暂魂态**（可选，可跳过）：
- 以魂体感知冥界/前世残影
- 只读回顾本世关键事件
- 结束后进入选胎界面

### 3.5 恶意杀害与投胎

被 NPC 或其他玩家 **恶意杀害** 时：
- **不加权**投胎补偿或优渥出生
- 仍按本世功德（及判定值扩缩后的池）决定可选投胎范围
- 杀害本身是命数，因果通过事件链与业力反馈，而非系统替死者「报仇式」升档

## 四、六道轮回

### 4.1 六道类型

| 轮回道 | 类型 | 功德要求 | 特点 |
|--------|------|----------|------|
| **天道** | 出生 | 大善+特殊机缘 | 直接成仙，超越轮回 |
| **人道** | 出生 | 善/中立 | 正常人生，可修炼 |
| **畜生道** | 出生 | 恶/中立 | 野兽形态，记忆保留片段 |
| **植物道** | 出生 | 中立+随机 | 灵植形态，千年化形 |
| **鬼道** | **仅死后进入** | 特殊死亡/执念 | 魂修形态，非初始随机种族 |
| **饿鬼道** | 死亡后 | 大恶 | 永久饥饿，痛苦轮回 |
| **地狱道** | 死亡后 | 大恶+随机 | 受刑N年，痛苦至极 |

> **鬼修**不作为账号创建或随机出生的初始种族，只能从死亡后的魂态/特殊条件进入。

### 4.2 轮回分配规则

```
最终判定值 + 当前功德 → 生成可选投胎池 → 玩家选择 → 物种/身份在选项内随机或指定 → 属性继承
```

| 判定值范围 | 默认开放轮回道（池内可选） | 说明 |
|------------|---------------------------|------|
| 0-19 | 畜生道、饿鬼道、地狱道 | 池子窄，高恶念者常仅剩劣道 |
| 20-49 | 畜生道、人道、植物道 | 人道可选但物种/身份偏差 |
| 50-79 | 人道、植物道、仙界候补 | 主流人道，偶有高品物种 |
| 80-99 | 人道、仙界、天道 | 高功德者可触及天道候补 |

> 上表为 **池子边界**；池内具体物种、身份、出生地仍受 **功德等级** 进一步过滤（见 4.2.1）。玩家只能在当前池内选择，不能越级选择未解锁选项。

#### 4.2.1 功德与投胎可选项

| 功德等级 | 功德值范围 | 可选轮回道 | 可选物种/身份上限 | 出生地品质 |
|----------|------------|------------|-------------------|------------|
| 大恶 | ≤ -5000 | 仅饿鬼道、地狱道、部分畜生道 | 劣等物种 | 贫瘠/险地 |
| 恶 | -4999 ~ -1000 | 畜生道为主，少量人道 | 普通野兽、流民等 | 较差 |
| 中立 | -999 ~ 999 | 人道、植物道、部分畜生道 | 凡人各阶层 | 普通 |
| 善 | 1000 ~ 4999 | 人道、植物道，解锁优质身份 | 含修士苗子、灵植等 | 较好 |
| 大善 | ≥ 5000 | 人道、植物道、仙界候补、天道候补 | 含灵体、谪仙等稀有物种 | 优渥 |

**选择规则**：
- 玩家从当前功德解锁的 **投胎池** 中选定一项（轮回道 + 物种档位 + 身份类型）
- 具体出生地、家族、初始属性在选定档位内随机生成
- 功德不足时，高阶选项 **灰显不可选**，不得付费或外力绕过

### 4.3 特殊轮回

| 特殊轮回 | 触发条件 | 效果 |
|----------|----------|------|
| **天道飞升** | 判定值95+且功德≥5000 | 直接成仙 |
| **地狱受刑** | 判定值5-且功德≤-5000 | 受刑10年游戏时间 |
| **植物化形** | 植物道轮回满千年 | 可化为人形 |
| **饿鬼解脱** | 饿鬼道中积满善功 | 可转世为人 |

### 4.6 初始属性生成公式

```
最终属性 = 基础属性 + 物种加成 + 功德继承 + 随机波动

基础属性: 根据物种设定的基准值
物种加成: 物种特有的属性加成（如灵兽灵力+15%）
功德继承: 上一世功德的30%转化为悟性
随机波动: ±10%的随机值
```

### 4.7 初始年龄与寿元随机

#### 4.7.1 初始年龄随机

| 轮回道 | 初始年龄范围 | 概率分布 |

| **畜生道** | 0-5岁（幼年期） | 均匀分布 |
| **植物道** | 0-100年 | 均匀分布 |

#### 4.7.2 寿元随机机制

```
初始寿元 = 物种基础寿元 × (1 + 功德修正 + 随机波动)

物种基础寿元: 凡人100岁, 灵兽500岁, 千年古树1000年
功德修正: 每1000功德 +5%寿元（最高+20%）

```

| 物种 | 基础寿元 | 寿元范围 |
|------|----------|----------|
| **普通人类** | 100岁 | 80-140岁 |
| **妖族混血** | 150岁 | 120-210岁 |

## 五、重生机制

### 5.1 身份生成

```
玩家选定投胎选项 → 轮回道确定 → 身份在档位内生成 → 初始属性分配 → 开启新一世（替换旧生命）
```

> 新一世生成后，上一世角色实体从世界中移除；账号回到 **单命制** 状态，仅保留灵魂层数据（功德继承、印记、档案）。

### 5.2 属性继承

| 继承类型 | 继承内容 | 继承比例 |
|----------|----------|----------|
| **功德继承** | 上一世功德 | 30% |
| **因果印记** | 特殊印记 | 100% |
| **天赋继承** | 特殊天赋 | 随机50% |
| **记忆碎片** | 前世记忆 | 随机触发 |

### 5.3 重生保护期
- **新手保护**：前30分钟不会被攻击
- **属性保护**：初始属性有保底

## 七、系统约束

### 7.1 轮回限制
- **单命制**：任意时刻仅 1 条活跃生命，**禁止并行**
- **最小轮回间隔**：1分钟（上一世结算完成后）
- **最大轮回次数**：无限制
- **特殊轮回冷却**：天道飞升后10轮回冷却
- **投胎选择**：必须在功德允许的池内选择；不得存在未解锁选项

### 7.2 功德范围
- **最小值**：-10000
- **最大值**：10000
- **衰减规则**：每日衰减1%

### 7.3 随机公平性
- 使用加密随机数生成器
- 确保概率分布均匀
- 防止作弊

## 八、与其他系统交互

### 8.1 与因果系统
- 因果值影响功德结算
- 因果印记继承

### 8.2 与生命体验系统
- 轮回道影响初始年龄
- 轮回道影响属性成长

### 8.3 与五感共鸣系统
- 不同轮回道感知能力不同
- 植物道有特殊感知

### 8.4 与情感羁绊系统
- 重要NPC关系跨轮回记忆
- 羁绊值影响轮回加权
