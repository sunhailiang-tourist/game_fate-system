# 因果修真世界 - 生命体验系统设计

## 一、系统概述

### 1.1 设计目标
生命体验系统是「因果修真世界」的核心玩法系统，旨在实现：
- **真实的生命周期**（生老病死）
- **年龄相关的属性变化**
- **修炼与年龄的平衡**
- **生存挑战**：活下去就是最大的任务

### 1.2 核心机制

```
出生 → 生存 → 成长 → 修行 → 衰老 → 死亡 → 轮回
    ↓      ↓      ↓       ↓       ↓       ↓       ↓
  随机形态  解决温饱  年龄增长  追求大道  能力衰退  生命终结  重新开始
```

### 1.3 无任务体系设计

**核心理念**：没有传统任务系统，**"活下去"就是唯一的任务**

**新手引导理念**：
> **一个自由的世界，唯一的目标就是活着，活下去，争取活的更好，全凭自己的能力与选择，帝王将相宁有种乎，神仙宗祖也未尝不可撼动...一切皆是命运。**

| 设计原则 | 说明 |
|----------|------|
| **自由探索** | 玩家可以自由决定如何生存 |
| **生存挑战** | 吃饭、喝水是每天必须解决的问题 |
| **真实风险** | 饿死、渴死是真实存在的风险 |
| **多样化人生** | 有的玩家很快活的很好，有的可能很快饿死 |
| **弱肉强食** | 野兽可以被猎杀获取资源 |
| **命运无常** | 一切皆是命运，全凭自己把握 |
| **无限可能** | 帝王将相宁有种乎，神仙宗祖也未尝不可撼动 |

### 1.4 生存等级

| 生存等级 | 描述 | 状态 |
|----------|------|------|
| **生存无忧** | 衣食无忧，资源充足 | 最佳状态 |
| **基本生存** | 能够解决温饱 | 正常状态 |
| **勉强维持** | 时有断顿，勉强生存 | 警告状态 |
| **濒临死亡** | 饥饿/口渴严重，生命垂危 | 危险状态 |
| **死亡** | 饿死/渴死 | 轮回 |

## 二、生命周期

### 2.1 生命阶段

| 阶段 | 年龄范围 | 描述 | 特点 |
|------|----------|------|------|
| **婴儿期** | 0-3岁 | 无法行动 | 完全依赖 |
| **童年期** | 4-12岁 | 开始学习 | 可塑性强 |
| **少年期** | 13-18岁 | 快速成长 | 精力充沛 |
| **青年期** | 19-35岁 | 实力巅峰 | 黄金时期 |
| **壮年期** | 36-55岁 | 经验丰富 | 资源充足 |
| **老年期** | 56-死亡 | 逐渐衰退 | 智慧巅峰 |

### 2.2 年龄增长机制
- **游戏时间1小时 = 现实时间1分钟**
- **年龄增长速度**：每游戏日增长1岁
- **寿命上限**：凡人100岁，修士根据境界提升

## 五、生存需求

### 5.1 基础需求（凡人特性）

| 需求类型 | 描述 | 衰减速度 | 后果 |
|----------|------|----------|------|
| **饥饿度** | 食物需求 | 每小时-10 | 生命值下降 |
| **口渴度** | 饮水需求 | 每小时-8 | 生命值下降 |
| **体力** | 行动消耗 | 行动消耗 | 无法行动 |
| **精力** | 精神需求 | 修炼/战斗消耗 | 无法修炼 |

### 5.2 需求满足

| 需求类型 | 满足方式 | 效果 |
|----------|----------|------|
| **饥饿度** | 食用食物 | 恢复饥饿度 |
| **口渴度** | 饮用清水/灵泉 | 恢复口渴度 |
| **体力** | 休息/打坐 | 恢复体力 |
| **精力** | 睡眠/冥想 | 恢复精力 |

### 5.3 辟谷机制（修士专属）

#### 5.3.1 辟谷等级

| 境界 | 辟谷能力 | 说明 |
|------|----------|------|
| **凡人** | 无 | 必须进食饮水 |
| **炼气期** | 初步辟谷 | 可减少50%饮食需求 |
| **筑基期** | 小辟谷 | 可减少80%饮食需求 |
| **金丹期** | 辟谷 | 无需进食，仍需饮水 |
| **元婴期** | 大辟谷 | 完全无需饮食饮水 |
| **化神期+** | 辟谷圆满 | 不食人间烟火 |

#### 5.3.2 辟谷效果

| 辟谷等级 | 饥饿度衰减 | 口渴度衰减 | 修炼效率 |
|----------|------------|------------|----------|
| 凡人 | 正常 | 正常 | 正常 |
| 炼气期 | -50% | -50% | +5% |
| 筑基期 | -80% | -80% | +10% |
| 金丹期 | 0 | -50% | +15% |
| 元婴期 | 0 | 0 | +20% |

### 5.4 境界提升要素

| 要素 | 影响权重 | 说明 |
|------|----------|------|
| **灵根质量** | 30% | 灵根纯度、属性数量 |
| **身体素质** | 25% | 体力、耐力、体质 |
| **悟性** | 20% | 领悟能力 |
| **功法品质** | 15% | 修炼功法等级 |
| **机缘运气** | 10% | 随机因素 |

## 六、身体素质极限系统

### 6.1 身体素质属性

| 属性 | 凡人上限 | 筑基上限 | 金丹上限 | 元婴上限 |
|------|----------|----------|----------|----------|
| **体力** | 100 | 200 | 500 | 1000 |
| **耐力** | 100 | 150 | 300 | 500 |
| **体质** | 100 | 150 | 300 | 500 |

### 6.2 极限突破机制

#### 6.2.1 突破流程

```
属性达到上限 → 持续10分钟 → 异常提示 → 选择继续/停止 → 结果判定
```

#### 6.2.2 异常提示内容
> "警告！您的[属性]已达到当前境界上限！继续突破极限将有极高风险！"

#### 6.2.3 突破结果判定

| 情况 | 突破概率 | 失败后果 |
|------|----------|----------|
| **普通体质 + 无特殊因素** | 5% | 暴毙（立即死亡） |
| **普通体质 + 有特殊因素** | 15% | 暴毙（立即死亡） |
| **特殊体质 + 无特殊因素** | 20% | 暴毙（立即死亡） |
| **特殊体质 + 有特殊因素** | 50% | 暴毙（立即死亡） |

#### 6.2.4 特殊体质列表

| 体质名称 | 效果 | 稀有度 |
|----------|------|--------|
| **混沌体** | 全属性突破概率+20% | 神话 |
| **先天道体** | 悟性+50%，突破概率+15% | 传说 |
| **九阳体** | 火属性亲和，突破概率+10% | 稀有 |
| **九阴体** | 冰属性亲和，突破概率+10% | 稀有 |
| **霸体** | 体力上限+100%，突破概率+5% | 普通 |

#### 6.2.5 特殊因素列表

| 因素 | 效果 | 获取方式 |
|------|------|----------|
| **天材地宝** | 突破概率+10% | 秘境探索 |
| **特殊功法** | 突破概率+5% | 奇遇获得 |
| **高人指点** | 突破概率+10% | NPC互动 |
| **天道眷顾** | 突破概率+15% | 高功德 |

### 6.3 突破成功效果

| 属性 | 突破后上限提升 |
|------|----------------|
| **体力** | +50% |
| **耐力** | +30% |
| **体质** | +40% |

## 七、自杀机制

### 7.1 自杀方式

| 方式 | 操作 | 冷却时间 |
|------|------|----------|
| **自刎** | 使用武器自杀 | 1小时 |
| **绝食** | 不进食直到饿死 | 无 |
| **跳崖** | 从高处跳下 | 无 |

### 7.2 自杀惩罚

| 惩罚类型 | 说明 |
|----------|------|
| **功德损失** | -500功德 |
| **天道标记** | +1标记（增加渡劫难度） |
| **轮回限制** | 自杀后投胎**必须随机**，无法选择 |
| **记忆清除** | 前世记忆完全清除 |

### 7.3 自杀后轮回规则

```
自杀死亡 → 灵魂审判 → 强制随机轮回 → 随机物种/身份 → 重新出生
    ↓              ↓                ↓               ↓
 无法选择      功德惩罚         随机分配         无继承
```

## 九、系统约束

### 9.1 年龄限制
- **最小年龄**：0岁
- **最大年龄**：根据境界
- **年龄增长**：不可逆转

### 9.2 需求限制
- **饥饿度上限**：100
- **口渴度上限**：100
- **体力上限**：根据境界
- **精力上限**：100

### 9.3 死亡条件
- **饥饿度归零**：每10秒损失5%生命值
- **口渴度归零**：每10秒损失5%生命值
- **生命值归零**：死亡
- **突破极限失败**：暴毙
