# 因果修真世界 - 五感共鸣系统设计

> **MVP 纳入**：五感系统为 MVP 核心沉浸体验之一，与生存、事件、战斗联动。

## 一、系统概述

### 1.1 设计目标
五感共鸣系统是「因果修真世界」的核心玩法系统，旨在实现：
- **多维度感知**世界
- **感知失误**的真实后果
- **修炼感知**的成长机制

### 1.2 核心机制

```
环境刺激 → 感官感知 → 判断处理 → 结果反馈
```

---

## 二、五感系统

### 2.1 五感类型

| 感官类型 | 感知内容 | 触发方式 | 反馈方式 |
|----------|----------|----------|----------|
| **视觉** | 环境、NPC、物品 | 被动 | 画面显示 |
| **听觉** | 声音、对话、危险预警 | 被动 | 声音播放 |
| **嗅觉** | 气味、药材、毒物 | 主动/被动 | 提示信息 |
| **触觉** | 材质、温度、陷阱 | 主动/被动 | 震动反馈 |
| **直觉** | 因果波动、命运警示 | 被动 | 特殊提示 |

### 2.2 五感属性

| 属性 | 描述 | 影响 |
|------|------|------|
| **感知范围** | 五感覆盖的距离 | 发现隐藏事物的距离 |
| **感知精度** | 分辨细节的能力 | 识别物品/陷阱的准确率 |
| **感知速度** | 处理感知信息的速度 | 反应时间 |
| **感知抗性** | 抵抗干扰的能力 | 抵抗幻术/毒药 |

---

## 三、感知机制

### 3.1 感知流程

```
环境数据 → 感官接收 → 大脑处理 → 决策输出 → 行动执行
```

### 3.2 感知判定

**感知成功率 = 基础成功率 + 感知属性加成 - 环境干扰**

| 环境因素 | 干扰值 | 说明 |
|----------|--------|------|
| 晴朗白天 | 0 | 最佳感知条件 |
| 阴天 | -5 | 轻微干扰 |
| 夜晚 | -15 | 明显干扰 |
| 雨天 | -10 | 中等干扰 |
| 雾天 | -20 | 严重干扰 |
| 战斗中 | -25 | 极大干扰 |

---

## 四、感知失误后果

### 4.1 失误类型

| 失误类型 | 后果 | 严重程度 |
|----------|------|----------|
| **轻度失误** | 错过隐藏物品 | 轻微 |
| **中度失误** | 触发陷阱受伤 | 中等 |
| **重度失误** | 中毒/重伤 | 严重 |
| **致命失误** | 死亡 | 致命 |

### 4.2 失误概率

**失误概率 = (100 - 感知精度) × 环境干扰系数**

| 失误概率 | 结果 |
|----------|------|
| 0-10% | 成功感知 |
| 11-30% | 轻度失误 |
| 31-60% | 中度失误 |
| 61-90% | 重度失误 |
| 91-100% | 致命失误 |

### 4.3 失误后果示例

| 场景 | 失误后果 | 具体表现 |
|------|----------|----------|
| 探索秘境 | 触发陷阱 | 受到伤害 |
| 采集药材 | 误采毒草 | 中毒 |
| 战斗 | 错过攻击时机 | 被敌人攻击 |
| 社交 | 误解NPC意图 | 好感度下降 |

## 七、系统约束

### 7.1 感知限制
- **最大感知范围**：100米
- **最小感知范围**：1米
- **感知冷却**：无冷却

### 7.2 失误保护
- **每日免死次数**：3次
- **失误保护期**：重生后5分钟

### 7.3 修炼限制
- **感知境界上限**：5级
- **每日修炼次数**：无限制

---

## 八、与其他系统交互

### 8.1 与因果系统
- 因果值影响感知精度
- 高因果值增加预感能力

### 8.2 与生命体验系统
- 年龄影响感知能力
- 老年期感知增强

### 8.3 与轮回系统
- 不同轮回道感知能力不同
- 植物道有特殊感知

### 8.4 与情感羁绊系统
- 感知能力影响NPC互动
- 高感知可发现NPC隐藏情绪
