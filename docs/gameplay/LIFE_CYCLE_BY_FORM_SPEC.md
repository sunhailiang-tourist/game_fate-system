# 因果修真世界 - 多形态生命规格

> **用途**：人/兽/植/鬼各形态的玩法与生命周期差异。  
> **版本**：v1.0 | 2026-06-12

---

## 一、形态总览

| 形态 | 进入方式 | 自然死亡期 | 可修炼 |
|------|----------|------------|--------|
| **人道** | 投胎池选择 | 按物种寿元 | 是 |
| **畜生道** | 投胎池选择 | 按物种寿元 | 灵兽可 |
| **植物道** | 投胎池选择 | 有（如干旱枯死） | 化形后可 |
| **鬼道** | **仅死后**进入 | 执念消散/渡化 | 魂修 |
| **饿鬼/地狱** | 大恶投胎 | 特殊 | 受限 |

> **鬼修不作初始随机种族**。

---

## 二、人道

### 生命周期
- 婴儿期（0-3）：**有感知、无行动**；可发生溺水等风险事件
- 童年→老年：见生命体验系统

### 核心玩法
- 生存需求（饥饿、口渴）
- 社会路径：农/商/仕/盗/修真等
- 完整距离战斗

---

## 三、畜生道

### 生命周期
- 幼年期短（0-5 游戏年）
- 寿元依物种：野兽 ~50 年，灵兽 ~500 年

### 核心玩法
| 能力 | 说明 |
|------|------|
| 捕猎 / 觅食 | 替代「购买食物」 |
| 领地 | 区域 hazard 与生态联动 |
| 灵兽修炼 | 达阈值可 **化形**（需渡劫） |
| 记忆 | 片段保留，影响下一世印记 |

### 身份模板
山林野兽、宗门灵兽、荒野凶兽、妖兽统领（见轮回系统 4.5.2）

---

## 四、植物道

### 生命周期
- 初始年龄 0-100 游戏年
- **自然死亡期**：如长期干旱 → 枯死风险（hazard 累积）

### 核心玩法
| 能力 | 说明 |
|------|------|
| 根系感知 | 五感偏触觉/嗅觉 |
| 枝桠互动 | 可庇护树阴下草木、鸟类 |
| 主动行为 | 非完全静止：照顾寄生鸟、释放花粉等 |
| 化形 | 百年~千年修为后渡劫化形 |
| 功德积累 | 庇护生灵等行为计入功德 |

### 设计意图
树并非不能动——有限主动行为决定功业与下一世池档位。

---

## 五、鬼道（仅死后）

### 进入条件
- 死亡后执念未散
- 特殊因果事件
- **非**账号创建或随机出生选项

### 核心玩法
| 能力 | 说明 |
|------|------|
| 魂体移动 | 无实体需求（不饥饿） |
| 附身/惊吓 | 有限交互，高业力风险 |
| 魂修 | 独特修炼路径，神通由魂历生成 |
| 解脱 | 积善/渡化 → 可转世入人道投胎池 |

### 与魂修短阶段区别
- **魂修短阶段**：死亡后过渡 UI，短时可跳过
- **鬼道轮回**：完整一世形态，有独立寿元（执念值）

---

## 六、跨形态共通规则

1. **单命制**：任意时刻仅一种形态一条命
2. **死亡** → 功德结算 → 选胎（形态在池内选择）
3. **神通**：均由 `经历→感悟` 生成，非技能树
4. **战斗**：距离体系万灵通用（见战斗系统）
5. **时间**：统一 1:10 比例

---

## 七、形态与投胎池映射

| 功德档位 | 典型可选形态 |
|----------|--------------|
| 大恶 | 饿鬼、地狱、劣等畜生 |
| 恶 | 畜生为主 |
| 中立 | 人、植、部分畜生 |
| 善 | 人、植、优质身份 |
| 大善 | 人、植、仙界候补 |

鬼道 **不出现在** 初始投胎池；仅死后分支。

| 优先级 | 形态 | 说明 |
|--------|------|------|
| P0 | 人道 | 核心验证 |
| P1 | 畜生道 | 简化捕猎/化形 |
| P1 | 植物道 | 有限互动 + 枯死 hazard |
| P2 | 鬼道完整一世 | 可先仅魂修短阶段 |
