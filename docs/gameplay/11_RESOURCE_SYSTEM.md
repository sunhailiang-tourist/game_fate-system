# 资源系统设计文档

---

## 一、核心设计原则

### 1.1 万物皆可食原则

**核心理念**：**飞禽走兽，树木花草，水里游的，天上飞的，只要打得过，都可以吃。如果没毒吃了就能活下去。**

| 原则 | 说明 |
|------|------|
| **万物可食** | 所有生物（飞禽、走兽、水生、植物）都可以尝试食用 |
| **打得过才能吃** | 需要击败目标才能获取食物 |
| **无毒即可生存** | 只要没毒，吃了就能活下去 |
| **后果自负** | 食用后的效果（有益/有害）由玩家自己承担 |
| **知识积累** | 通过尝试积累食物知识 |

### 1.2 可食用生物分类

| 类别 | 示例 | 食用方式 | 注意事项 |
|------|------|----------|----------|
| **飞禽** | 鸡、鸭、鹅、鹰、孔雀 | 击杀后烹饪食用 | 部分有羽毛需要处理 |
| **走兽** | 牛、羊、猪、鹿、狼、虎 | 击杀后烹饪食用 | 大型野兽需要更强实力 |
| **水生生物** | 鱼、虾、蟹、龟、蚌 | 捕捞后烹饪食用 | 需要水域环境 |
| **植物/花草** | 草、果实、根茎、花朵 | 直接采摘食用 | 部分有毒 |
| **灵植** | 灵草、仙草、仙药 | 直接食用或炼丹 | 可能有特殊效果 |
| **灵兽** | 有灵智的野兽 | 击杀后食用 | 实力较强 |

### 1.3 食用效果分类

| 效果类型 | 说明 | 示例 |
|----------|------|------|
| **有益** | 恢复生命/体力/精力 | 食用灵草恢复灵力 |
| **中性** | 无明显效果 | 食用普通野草 |
| **有害** | 造成伤害/中毒 | 食用毒草中毒 |
| **特殊** | 获得特殊效果 | 食用仙草获得增益 |
| **生存保障** | 无毒食物恢复基础生存需求 | 食用普通兽肉恢复饥饿度 |

### 1.3 弱肉强食机制

| 机制 | 说明 |
|------|------|
| **猎杀野兽** | 可以猎杀野兽获取材料 |
| **材料利用** | 野兽的内丹、血液、骨头都可以买卖和利用 |
| **内丹制药** | 野兽内丹可以用于炼药 |
| **骨头炼器** | 野兽骨头可以用于炼器 |
| **市场交易** | 野兽材料可以在市场交易 |

## 七、资源消耗与再生机制

### 7.1 消耗规则

| 资源类型 | 消耗方式 | 消耗后效果 |
|----------|----------|------------|
| 灵石 | 修炼/交易/打造 | 减少灵石数量 |
| 药材 | 炼丹/直接服用 | 药材消失 |
| 材料 | 炼器/炼丹 | 材料消失 |
| 秘籍 | 学习 | 秘籍消失（绑定） |

### 7.2 再生机制

| 资源类型 | 再生周期 | 再生条件 |
|----------|----------|----------|
| 灵草 | 1-1000年 | 未被破坏 |
| 矿脉 | 10-100年 | 未枯竭 |
| 灵兽 | 2-50年 | 自然繁衍 |
| 天材地宝 | 100-1000年 | 特殊条件 |

### 7.3 资源恒定法则

```
资源总量 = 初始总量 + 再生量 - 消耗量

当资源总量低于警戒值时：
- 再生速度加快
- 新资源点生成
- 秘境刷新
```
