# 因果修真世界 - 情感羁绊系统设计

## 一、系统概述

### 1.1 设计目标
情感羁绊系统是「因果修真世界」的核心玩法系统，旨在实现：
- **深度情感连接**的NPC互动
- **动态变化**的关系网络
- **跨轮回**的情感传承

### 1.2 核心机制

```
互动 → 好感度变化 → 关系升级 → 特殊效果 → 情感传承
```

---

### 2.1 羁绊分类

## 三、羁绊机制

### 3.1 好感度计算

**好感度变化 = 互动效果 + 时间衰减 + 事件修正**

| 互动类型 | 好感度变化 | 说明 |
|----------|------------|------|
| **赠送礼物** | +5-20 | 根据礼物价值 |
| **应其求助/共历事件** | +10-30 | 根据事件重量 |
| **日常聊天** | +2-5 | 每日限1次 |
| **帮助NPC** | +10-20 | 根据帮助内容 |
| **攻击NPC** | -20-50 | 严重下降 |
| **忽视NPC** | -1-5/天 | 时间衰减 |

### 3.2 羁绊建立流程

```
相遇 → 互动 → 好感度积累 → 羁绊确认 → 关系升级
```

### 3.3 羁绊事件

| 事件类型 | 触发条件 | 效果 |
|----------|----------|------|
| **羁绊升级** | 好感度达到阈值 | 解锁新互动 |
| **羁绊危机** | 好感度下降到阈值 | 关系破裂 |
| **羁绊事件** | 羁绊等级达到要求 | 状态触发的特殊事件 |
| **羁绊传承** | 玩家死亡/轮回 | 跨轮回记忆 |

## 五、羁绊传承

### 5.1 跨轮回传承

```
当前轮回羁绊 → 羁绊值积累 → 轮回结算 → 下一世继承
```

### 5.2 传承规则

| 羁绊类型 | 传承比例 | 说明 |
|----------|----------|------|
| **师徒缘** | 50% | 继承师父记忆 |
| **兄弟缘** | 40% | 继承兄弟情谊 |
| **情缘** | 60% | 继承前世情缘 |
| **宿敌缘** | 70% | 继承宿敌仇恨 |
| **道友缘** | 30% | 继承道友关系 |

### 5.3 羁绊记忆

| 记忆类型 | 触发条件 | 效果 |
|----------|----------|------|
| **模糊记忆** | 传承比例>0 | 偶尔想起 |
| **清晰记忆** | 传承比例>50% | 经常想起 |
| **完整记忆** | 传承比例>80% | 完全记得 |

## 六、系统约束

### 6.1 羁绊限制
- **最大羁绊数量**：10个
- **最大情缘数量**：1个
- **羁绊等级上限**：6级

### 6.2 好感度范围
- **最小值**：-100
- **最大值**：100
- **初始值**：0

### 6.3 时间衰减
- **每日衰减**：好感度 × 1%
- **衰减下限**：-50
- **特殊保护**：羁绊等级>3不受衰减影响

## 七、与其他系统交互

### 7.1 与因果系统
- 羁绊关系影响因果值
- 高好感度增加善缘

### 7.2 与生命体验系统
- 年龄影响羁绊建立
- 老年期羁绊更容易传承

### 7.3 与轮回系统
- 羁绊记忆跨轮回传承
- 羁绊值影响轮回加权

### 7.4 与五感共鸣系统
- 高感知可发现NPC好感度
- 可察觉NPC隐藏情绪
